# Props

# gap

  • Type: String or Array<String>
  • Default: 0

A responsive prop bound to the spacing scale that applies a specified space between successive items in the <HorizontalScroll />.

# as

  • Type: String
  • Default: div

The HTML element to render the < Inline /> component as. See semantic HTML for more details.

# css

  • Type: Object
  • Default: {}

Additional styles applied to the component that leverage the theme when available. Visit the css prop documentation for more details.
