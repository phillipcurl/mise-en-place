# Props

# as

  • Type: String
  • Default: p

The HTML element to render the <BaseText /> component as. See semantic HTML for more details.

# maxWidth

  • Type: String or Array<String>
  • Default: var(--measure)

A responsive prop that specifies the max-width CSS value for the <BaseText /> component. By default, this value is set to the measureWidth value in your mise-en-place config.

# lineHeight

  • Type: String
  • Default: var(--line-height-body)

The line-height CSS value for the text. By default, this value is set to the lineHeightBody value in your mise-en-place config.
